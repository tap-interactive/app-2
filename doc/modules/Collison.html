checkIfOverLaping (slob_num)
Check overlaping The function detects if a the Blob overlapse the Slob entity, using the logic describe in the pictures below. Briefly the top left corner and the bottom right corners of both the Slob and the blob is used to determined collision. The ID of the Slob is inputted to the function and is used to detect overlaping. If overlaping is detected a boolean parameter will be returned.

Parameters:

  • slob_num The input parameter representing the ID of the Slob.

Returns:

    bool
