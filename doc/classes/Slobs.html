Slobs:randomPlace (sb_type, lvl_slob)
Random spawning place The spawning handler for the slobs making them spawn at the edges of the screen and at random places each time.

Parameters:

  • sb_type The input parameter that represents the type of Slob to be spawned.
  • lvl_slob The input parameter that represents the level of the Slob to be spawned.

Returns:

  1. window_width - tab_level[lvl_slob] The output representing the spawn width.
  2. math.random(side_margin,window_height - tab_level[lvl_slob]) The output representing the spawn height.

Slobs:getCharacteristicNumberSlobs (num)
Slob characteristics The function handling the parameters assigned to in a correct way display a Slob of different levels.

Parameters:

  • num The input parameter representing the ID of the Slob to get the characteristics from.

Returns:

  1. (slob_list[num].x + slob_list[num].width/4) The output parameter representing the x-coordinate for the Slob.
  2. (slob_list[num].y + slob_list[num].height/2) The output parameter representing the y-coordinate for the Slob.
  3. slob_list[num].width/2 The output parameter representing the width to be displayed.
  4. slob_list[num].height/3 The output parameter representing the height to be displayed.
